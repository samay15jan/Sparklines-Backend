## Local development

### Prerequisites

- NodeJS v16+

### Run locally

- Clone the Repo

  ```sh
  git clone https://github.com/sanay15jan/sparklines-backend

  cd sparklines-backend
  ```

- Install the dependencies

  ```sh
  npm install
  ```

- Start development server

  ```sh
  npm run dev
  ```

## Host your own instance
